# CodeNest Hero

A responsive, high-end hero section built with React, Tailwind CSS, hls.js, and lucide-react.

## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

The background uses the supplied Mux HLS stream. `hls.js` is initialized with `enableWorker: false`, with native HLS fallback for Safari.
